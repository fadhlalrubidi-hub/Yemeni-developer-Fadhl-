# 🚀 المنصة السيبرانية للمطور فضل الشعيبي
**Worldwide Tech Domination Engine V4.6**

هذا المستودع يحتوي على الشفرة المصدرية للمنصة الشخصية والمحفظة الرقمية للمطور **فضل عبدالله مثنى الشعيبي**، أول مطور يمني يقود مشاريع تقنية عملاقة موجهة للمستقبل. تم بناء المنصة لتعكس رؤية استثمارية وهندسية فائقة التطور، مع تصميم يعتمد على الطابع "السيبراني" (Cyberpunk) وتقنية الزجاج (Glassmorphism).

---

## ✨ المميزات الحصرية (Features)

- 🔐 **نظام ولوج سيبراني:** شاشة تسجيل دخول تفاعلية لحماية الواجهة لا يمكن عبورها إلا ببيانات المطور المركزي، مع رسائل ترحيب "ملكية".
- ⏳ **مؤقت مائي زجاجي:** شاشة تحميل (Loader) مبتكرة بتصميم كرة زجاجية مائية وعداد تنازلي يمهد لدخول الموقع.
- 📱 **نظام توجيه فائق للجوال:** واجهة متجاوبة بالكامل بنسبة 100% مع الشاشات الصغيرة، وتتضمن "مؤشر إرشاد ذكي" (Smart Hint) متحرك لتوجيه الزوار لزر القائمة.
- ⚡ **محرك صفحات داخلي (SPA):** تنقل سلس وفوري بين أقسام الموقع دون الحاجة لإعادة تحميل الصفحة.
- 🎵 **تأثيرات صوتية מبرمجة (Audio Engine):** أصوات تفاعلية مدمجة عبر `AudioContext` تعمل عند النقرات، تسجيل الدخول، وأثناء العد التنازلي.
- 📰 **مكتبة مقالات ديناميكية:** نظام عرض مقالات يتيح التركيز (Focus Mode) على المقال المختار وتوسيع المحتوى بطريقة سينمائية.

## 🛠️ التقنيات المستخدمة (Tech Stack)

تم بناء هذه المنصة كملف واحد متكامل (Single-File Architecture) لضمان أقصى سرعة أداء، باستخدام التقنيات الأساسية (Vanilla):
- **HTML5:** الهيكلة الدلالية المتقدمة.
- **CSS3:** حركات مخصصة (Keyframes)، متغيرات CSS، وتأثيرات الـ Backdrop-filter.
- **JavaScript (Vanilla):** إدارة حالة الصفحات، التوقيت، محرك الصوت (Web Audio API)، وتأثيرات الكتابة التلقائية (Typewriter).

## 🚀 طريقة التشغيل (How to Run)

المشروع لا يتطلب أي خوادم معقدة:
